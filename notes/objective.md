I completed a Home SOC Lab project to strengthen my cybersecurity and SOC analyst skills.

In this lab, I built a mini security monitoring environment using Splunk, Windows logs, Sysmon, and VirtualBox. I configured log forwarding from a Windows VM to Splunk, generated test security events, and created detections for failed logins, successful logins, new user creation, PowerShell activity, and suspicious processes.

Skills practiced:
- SIEM monitoring
- Windows Event Log analysis
- Sysmon configuration
- Threat detection
- Incident investigation
- SOC analyst workflow

This project helped me understand how real SOC teams collect logs, detect suspicious behavior, and investigate alerts.
